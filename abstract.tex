Más práticas de programação são normalmente associadas à anti-patterns e cheiros de código. Muitos estudos já foram feitos sobre anti-patterns e cheiros de código em projetos orientados a objetos, mas poucos estudos foram realizados sobre este mesmo tema no contexto de aplicativos Android nativos. Será que esse contexto apresenta más práticas de programação específicas? Este estudo investigou a percepção de desenvolvedores quanto a existência de boas e más práticas de programação em aplicativos Android nativos através de um estudo qualitativo com 45 participantes de mais de 17 países diferentes. Neste artigo nós compartilhamos suas opiniões e experiências. De acordo com eles, há sim implementações específicas à aplicativos android que são consideradas más práticas e devem ser evitadas, bem como boas práticas recomendadas para mitigar essas más práticas. \\


\noindent \textbf{Keywords:} android, good and bad practices, code smells, code quality, code anomalies.
